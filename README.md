# Die-Pipeline
Discord bot to keep track of what movies you watch with your friends.
This small Discord bot owe's it's existance to a daily movie night combined wiuth personal boredom, etc.

# Usage:
1. Create a Discord bot and copy it's token.
2. Add that token to the auth.json file
3. Get a "The Movie Database" API key.
4. Paste API key into auth.json
5. Run node ./bot.js
6. Add the bot to your server and enjoy

# Features:
* Keep track of all the movies /series you and your friends watch
* See which actors / directors appear most in your movies
* Automatic list updating
* and more.
